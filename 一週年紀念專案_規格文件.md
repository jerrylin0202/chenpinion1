## 模組一:互動式週年卡片

### 1. 專案目標
製作一個給女友的專屬網頁,以可達鴨&吉利蛋為主角,呈現兩人一週年的故事與心意,透過點擊互動觸發動畫與音效,營造驚喜感。

### 2. 技術規格
- **形式**:單頁網站(HTML/CSS/JS,或 React 單頁元件)
- **部署**:靜態網站,部署到免費託管服務(如 GitHub Pages / Vercel / Netlify),產生一個專屬網址私訊分享給女友
- **存取控制**:不設密碼鎖,直接開啟即可瀏覽
- **適配裝置**:以**桌面瀏覽器為主**設計(目標螢幕約 27 吋 / 2K 解析度),手機則以**橫向瀏覽**為輔助適配
- **音效**:背景音樂(可選)+ 互動音效(點擊回饋音)

### 3. 互動流程設計

目前已實作的頁面結構與互動(框架完成,內容待填):

頁面結構為翻頁繪本式,共 5 頁:封面 → 回憶頁 01/02/03 → 結尾告白頁。桌面以左右箭頭/鍵盤方向鍵/底部圓點導覽,手機橫向支援滑動翻頁。

**角色情境化**:因可達鴨與吉利蛋為受版權保護角色,無法用 AI 重新生成新插圖,故採「既有去背角色 + CSS 場景」的情境化做法。每個回憶頁的角色會放在一個有機形狀的「情境舞台」中,搭配不同背景氛圍(pond 水塘綠 / blossom 花漾粉 / sunny 陽光黃),角色帶輕微浮動動畫。

**拍立得手帳風照片系統**(本版重點):每個回憶頁可放置多張「拍立得照片」,白色相紙外框、底部手寫字幕(Caveat 手寫字體)、隨機輕微傾斜營造散落手帳感,頂部有圖釘裝飾。照片數量可自由擴充(版面自動換行排列),點擊任一張照片會以燈箱放大檢視。目前皆為佔位卡,於程式底部 `MEMORIES` 設定區把照片的 `src` 換成真實照片(網址或 base64)、`cap` 改成手寫小語即可。

**互動效果**:點擊封面主視覺/角色舞台 → 音效+愛心彩蛋;點擊拍立得 → 燈箱放大;結尾頁點擊「點我看訊息」→ 愛心雨+告白訊息卡片滑入。全程背景有上飄氛圍泡泡與貼紙裝飾(愛心/星星/驚嘆號)。

### 4. 角色與美術風格
- **角色**:可達鴨(男方視角)、吉利蛋(女方視角)— 接近原作配色與外型
- **畫風**:✅ **已確認採用「日系貼圖可愛風」**,參考特徵如下:
  - 粗黑線稿(thick outline),貼紙白色描邊(die-cut sticker 效果)
  - 扁平飽和色塊,不做複雜光影
  - 表情生動誇張(瞇眼笑、驚訝睜大眼等)
  - 周圍點綴愛心 ♥、星星 ✦、驚嘆號「!!」等貼紙元素增加活潑感
- **產出方式**:使用 AI 圖像生成製作多張情境插畫(開場、時間軸節點、互動彩蛋、結尾告白場景),統一套用上述貼紙風格特徵

## 模組二:每月手機桌布自動生成與寄送

### 1. 專案目標
每月自動產生一張以可達鴨&吉利蛋為主題的 iPhone 桌布,內容結合「該月節日/季節主題」+「台灣國定假日標註」+「交往天數倒數/紀念資訊」,並自動寄送到女友信箱。

### 2. 系統架構(建議方案:Google Apps Script,免費)

```
[Google Apps Script 專案]
  │
  ├─ 時間驅動觸發器(每月1日 09:00 自動執行)
  │
  ├─ Step 1:讀取當月資訊
  │     - 當月月份、季節
  │     - 台灣當月國定假日/節日列表(維護一份月曆資料表)
  │     - 計算交往天數/週年倒數
  │
  ├─ Step 2:生成桌布圖片
  │     - 方案A:套用「固定版型 + 動態文字」(用 Google Slides API 製作模板,
  │              程式自動替換文字後輸出圖片)
  │     - 方案B:呼叫 AI 圖像生成 API,依當月主題生成可達鴨&吉利蛋插畫,
  │              再用程式疊加文字資訊
  │     - 建議:兩者結合 — 每月用 AI 生成「當月主題插畫」(共12張可預先批量
  │              製作),再用 Apps Script 動態疊加當月日期/倒數文字
  │
  ├─ Step 3:輸出圖片
  │     - 尺寸:1179 x 2556 px(iPhone 15 鎖定畫面比例)
  │     - 格式:PNG
  │
  └─ Step 4:寄送 Email
        - 使用 GmailApp.sendEmail() 寄送至女友信箱
        - 附件:當月桌布圖片
        - 信件內文:簡短訊息(可固定模板,如「OO月的桌布來囉~」)
```

### 3. 桌布內容規格
| 元素 | 說明 |
|---|---|
| 主視覺 | 可達鴨&吉利蛋當月主題插畫(依季節/節日延伸,例如:過年穿新衣、情人節、中秋賞月、聖誕節等) |
| 節日標註 | 當月台灣國定假日/重要節日文字(例如:228和平紀念日、清明節、端午節等) |
| 紀念資訊 | 交往天數累計 或 距離下次週年倒數天數 |
| 尺寸 | 1179 x 2556 px(iPhone 15) |

> 待確認:12 個月的插畫主題是要一次性預先規劃(例如 1月過年/2月情人節/.../12月聖誕),還是要保留彈性、每月臨時決定?建議**先一次性規劃好全年12個主題**,降低每月維護成本。

### 4. 自動化排程設計
- **執行頻率**:每月1日自動執行一次
- **執行環境**:綁定在你的 Google 帳號下的 Apps Script,免費額度足夠此用量
- **失敗通知**:建議加上錯誤通知機制(若當月執行失敗,寄一封通知信到你自己的信箱)
- **停止機制**:可設定結束日期(例如交往滿2年後自動停止,或手動關閉觸發器)

### 5. 所需素材清單
- [ ] 女友的 Email 信箱
- [ ] 兩人交往紀念日日期(用於倒數計算)
- [ ] 全年12個月主題插畫(可達鴨&吉利蛋,依台灣節日/季節)
- [ ] 桌布版型設計(文字位置、字型、配色)
- [ ] 台灣國定假日資料(可用公開資料來源,每年需更新一次)

## 模組三:數位相框 + Google Drive 自動同步

### 1. 建議型號(支援 Google Drive 直接同步)

市面上多數熱門品牌(如 Aura)**不支援 Google Drive**,只支援 Google 相簿。目前已知支援 Google Drive 同步的型號:

| 品牌/型號 | 特點 | 備註 |
|---|---|---|
| **Pix-Star** 系列(8吋/10吋/15吋 WiFi 數位相框) | 支援多種雲端來源同步,包含 **Google Drive、Dropbox、Google 相簿、Flickr** 等;支援離線檢視已同步相片 | 目前已知少數可直接連 Google Drive 資料夾的品牌,建議優先考慮 |
| Nixplay 系列 | 支援 Google 相簿(需訂閱 Plus 才完整使用),**不支援 Google Drive** | 不符合你的需求,僅列為對照 |
| Aura 系列 | 支援 Google 相簿、iCloud,**官方明確不提供 Google Drive 整合** | 不符合你的需求,僅列為對照 |

> 建議:以 **Pix-Star** 為主要選購方向。實際在台灣的購買管道(代購/海外購)、尺寸與當前售價需要另外查詢確認,我可以再協助找台灣可購買的賣場連結與當前價格。

### 2. 設定流程(概念)
1. 在 Google Drive 建立一個專屬資料夾(例如「給OO的回憶相簿」)
2. 將相框與該 Google 帳號的 Drive 資料夾連結同步
3. 之後只要把新照片丟進該資料夾,相框會自動更新顯示

### 3. 注意事項
- Google 雲端硬碟桌面版的「相簿備份」功能在 2026 年中將陸續調整,但**Drive 資料夾本身的同步功能不受影響**,Pix-Star 走的是直連 Drive 資料夾,屬於不同機制,目前看來不受此變動影響
- 建議先確認相框購買管道與台灣保固/客服支援情況
